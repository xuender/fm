# File Meta (FM)

[![Go Report Card](https://goreportcard.com/badge/github.com/xuender/fm)](https://goreportcard.com/report/github.com/xuender/fm)

Move file based on meta.

## Install

```shell
go install github.com/xuender/fm@latest
```

## Init

Init FM config.

```shell
cd dir
fm init
```

## Meta

Show file meta.

```shell
fm meta [file or dir]
```

## Move

Move file based on meta.

```shell
cd dir
fm
```
